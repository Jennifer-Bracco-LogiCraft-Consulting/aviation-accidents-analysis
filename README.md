# Aviation Accidents Analysis (1919–2023)

This project focuses on the analysis of aviation accidents from 1919 to 2023 using Python.

## Objective
Transform a large historical dataset into structured information to understand trends over time, geographical distribution and differences in incident severity.

## Data
The dataset contains information about aviation accidents, including date, aircraft type, operator, location, country and number of fatalities.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Plotly Express
- country_converter

## Results
- Evolution of aviation accidents over time
- Analysis of incidents related to criminal acts and terrorism after September 11, 2001
- Comparison between number of incidents and number of fatalities
- Identification of countries with the highest number of accidents
- Differences in incident severity across operators
- Analysis of aircraft types associated with the highest number of fatalities
- Evaluation of data quality over time
